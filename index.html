HTML

HTML stands for hypertext markup language.
It creates what you are able to see (and not see) on websites.

Elements

HTML elements help make up a web page and enable it to act in distinct ways.

Tags

Tags manipulate the text/code to enable it to become bold, to add an
image or by adding links to a website.
Some tags are considered to be void because they do not need
a closing tag.

Inline or Block?

Block elements differ from inline elements because they put a "box" around its content.

Span or Div?

What makes these two elements different is that span is inline while div are block.

What I Learned from Stage 1/2

How to Create a Structured Document

To create a structured document, you need HTML, CSS and Javascript. Each of these create the webpage and allow it to do different things. HTML is the content, CSS is the style of the content and Javascript makes webpages interactive for the user.

Boxes

Webpage structure is similar to a tree. Each element can have other elements that "Branch" off and have elements within them. However, these elements are contained in boxes that separate each element from one another.

Text Editors

Text editors are used by programmers to help write code easier. Many programs, such as Sublime Text, can create closing tags when writing an opening tag. It can also help indicate where you might've made a mistake by changing the colors of the text.

What I learned from Stage 3

CSS

CSS stands for Cascading Style Sheets. In CSS, the order of the code matters, since the top most code overrules all. Inheritance helps prevent repitition in the overall code.

Box Model

  • Center: Content - what is appearing on the website
  • Padding: clears an area around content, affected by background color
  • Border: inherited from color property of box
  • Margin: transparent - space between boxes

Element width = Border+Padding+content+Padding+border

Code/test/refine

  1. Look for natural boxes
  2. Look for repeated styles/elements
  3. Write HTML
  4. Apply styles (Biggest to smallest)
  5. Fix things - look on other browsers
